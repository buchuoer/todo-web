# Minimus 项目 - 优化建议清单

## ✅ 已完成的修复

### 1. 分类选择不再立即提交
- **问题**: 点击分类会直接创建 todo
- **修复**: 分类选择仅设置状态，点击 + 按钮或按 Enter 时才提交
- **改进**: 分类按钮显示当前选中的分类指示器

### 2. 文字可读性提升
- **分类标签**: 字体 11px → 12px，加粗，增加边框
- **日志日期**: 添加背景渐变，字体加粗，增加 padding
- **筛选按钮**: 字体加大，增加阴影效果
- **分类选择弹窗**: 添加选中状态样式（蓝色背景）

## 💡 更多优化建议

### 🎯 操作体验优化

#### 1. 拖拽排序
- **功能**: 拖拽调整待办顺序
- **实现**: 使用 @dnd-kit 或 react-beautiful-dnd
- **价值**: 用户可自定义优先级

#### 2. 批量操作
- **功能**: 多选待办进行批量删除/完成/修改分类
- **实现**: 添加复选框 + 批量操作栏
- **价值**: 提高大量待办的管理效率

#### 3. 待办编辑
- **功能**: 点击待办文字可直接编辑
- **实现**: 类似日志的编辑功能
- **价值**: 快速修改，无需删除重建

#### 4. 撤销操作
- **功能**: 删除/完成待办后显示撤销按钮（3秒）
- **实现**: 使用 setTimeout + Toast 提示
- **价值**: 防止误操作

#### 5. 快捷操作
- **功能**: 键盘快捷键（如 Ctrl+N 新建，Ctrl+数字切换标签）
- **实现**: 全局键盘监听
- **价值**: 提升效率

---

### 🎨 布局设计优化

#### 1. 响应式布局
- **现状**: 仅适配移动端底部导航
- **优化**:
  - 桌面端侧边栏布局
  - 平板端双列布局（待办列表 + 详情）
  - 更好的大屏适配

#### 2. 空状态设计
- **现状**: 简单的"暂无"文字
- **优化**:
  - 添加插图/动画
  - 引导用户创建第一个待办
  - 示例内容一键导入

#### 3. 过渡动画
- **功能**: 添加/删除/完成待办时的平滑动画
- **实现**: CSS transition + Framer Motion
- **价值**: 提升视觉体验

#### 4. 暗黑模式
- **功能**: 支持亮色/暗色主题切换
- **实现**: CSS 变量 + localStorage 保存偏好
- **价值**: 护眼，现代应用标配

#### 5. 分类颜色
- **功能**: 不同分类使用不同颜色
- **实现**:
  ```
  工作 = 蓝色
  学习 = 绿色
  生活 = 橙色
  ```
- **价值**: 视觉区分，快速识别

---

### 🔧 功能丰富

#### 1. 待办优先级
- **功能**: 高/中/低优先级标记
- **实现**: 添加优先级字段 + 颜色标识
- **价值**: 重要事项突出显示

#### 2. 子任务
- **功能**: 待办可添加子任务清单
- **实现**: 嵌套数据结构 + 进度显示
- **价值**: 复杂任务分解

#### 3. 重复待办
- **功能**: 每日/每周/每月重复
- **实现**: 添加重复规则字段
- **价值**: 习惯追踪，定期任务

#### 4. 待办搜索
- **功能**: 搜索待办内容
- **实现**: 顶部搜索框 + 实时过滤
- **价值**: 大量待办时快速定位

#### 5. 数据统计
- **功能**: 统计完成率、分类分布等
- **实现**: 图表（饼图、柱状图）
- **价值**: 可视化生产力

#### 6. 数据导入导出
- **功能**: 导出 JSON/CSV，导入恢复
- **实现**: File API + 下载链接
- **价值**: 数据备份，迁移

---

### 🤖 AI 功能增强

#### 1. 智能分类
- **功能**: AI 自动建议待办分类
- **实现**: 输入时调用 AI 分析文本
- **价值**: 减少手动操作

#### 2. 智能截止日期
- **功能**: AI 从文本提取时间信息
- **示例**: "明天完成报告" → 自动设置明天
- **价值**: 自然语言输入

#### 3. 待办总结
- **功能**: AI 总结今日/本周待办完成情况
- **实现**: 定期生成报告
- **价值**: 复盘与规划

#### 4. 智能提醒
- **功能**: AI 根据截止日期和优先级建议提醒时间
- **价值**: 不再错过重要事项

---

### 📱 移动端优化

#### 1. 手势操作
- **功能**: 左滑完成，右滑删除
- **实现**: 触摸事件监听
- **价值**: 移动端高效操作

#### 2. 下拉刷新
- **功能**: 下拉同步数据（如果未来有云端）
- **实现**: Pull-to-refresh 组件

#### 3. PWA 支持
- **功能**: 添加到主屏幕，离线使用
- **实现**: Service Worker + Manifest
- **价值**: 原生应用体验
